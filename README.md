# 🚀 Svelte + Express + PostgreSQL on Azure

A full-stack web application deployed on Microsoft Azure with:
- **Frontend**: Svelte (with Vite)
- **Backend**: Express.js
- **Database**: PostgreSQL (Azure Flexible Server)
- **Deployment**: Azure App Service + GitHub Actions

## 🛠️ Tech Stack

- **Frontend**: Svelte, Vite, Axios
- **Backend**: Node.js, Express, PostgreSQL (pg)
- **Cloud**: Azure App Service, Azure Database for PostgreSQL
- **CI/CD**: GitHub Actions
